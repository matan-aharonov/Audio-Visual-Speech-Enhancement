# Audio-Visual-Speech-Enhancement
Engineering Project

### mediaio
A python wrapper for reading, writing and manipulating audio and video files.

#### Dependencies
* python >= 2.7
* scipy >= 0.19.0
* numpy >= 1.12.1
* imageio >= 2.1.2
* opencv >= 3.2.0

### face-detection
A python wrapper for detecting face and mouth regions in images using dlib library.

#### Dependencies
* python >= 2.7
* numpy >= 1.12.1
* dlib >= 19.4.0
* opencv >= 3.2.0
